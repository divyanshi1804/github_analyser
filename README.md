# GitHub Profile Analyzer API

A backend API that fetches GitHub user profiles and stores insights in MySQL.

## Tech Stack
- Node.js
- Express.js
- MySQL
- GitHub Public API
- Axios

## Setup Instructions

1. Clone the repository
```bash
git clone https://github.com/divyanshi1804/github_analyser.git
cd github_analyser
```

2. Install dependencies
```bash
npm install
```

3. Create a `.env` file
